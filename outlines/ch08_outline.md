# Chapter 8 Outline: Launch Files and Parameters

## Learning Objectives
- Understand the purpose of launch files.
- Write a simple launch file to start multiple nodes.
- Use parameters to configure nodes at runtime.
- Use a parameter file to store and load parameters.

## Sections
1.  **The Need for Launch Files**: Automating Node startup
2.  **Writing a Launch File**: The Python-based launch system
3.  **Starting Multiple Nodes**: The `Node` action
4.  **Parameters**: Configuring your nodes
5.  **Parameter Files**: Storing and loading parameters in YAML

## Labs
- **Lab 8.1**: Creating a launch file to start a talker and listener.
- **Lab 8.2**: Using parameters to change the message published by the talker.
