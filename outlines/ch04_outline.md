# Chapter 4 Outline: rclpy Programming

## Learning Objectives
- Create a ROS 2 package for Python.
- Write a simple `rclpy` node.
- Create publishers and subscribers.
- Use timers for periodic tasks.
- Understand the role of executors.

## Sections
1.  **Creating a Python Package**: The `package.xml` and `setup.py` files
2.  **Writing a Node**: The `rclpy.node.Node` class
3.  **Creating a Publisher**: Sending messages
4.  **Creating a Subscriber**: Receiving messages
5.  **Using Timers**: Triggering callbacks
6.  **Executors**: Spinning your nodes
