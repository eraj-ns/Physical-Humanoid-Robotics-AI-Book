# Chapter 7 Outline: URDF for Humanoids

## Learning Objectives
- Understand the basic structure of a URDF file.
- Create a simple humanoid robot model.
- Use the `joint_state_publisher` and `robot_state_publisher` to visualize the model in RViz2.
- Understand the difference between fixed and movable joints.

## Sections
1.  **The `<robot>` tag**: The root of the model
2.  **`<link>` tags**: Defining the robot's body parts
3.  **`<joint>` tags**: Connecting the links
4.  **Visualizing the Model**: Using RViz2
5.  **Joint Types**: `fixed`, `revolute`, `continuous`, `prismatic`

## Labs
- **Lab 7.1**: Creating a simple URDF model of a humanoid robot.
- **Lab 7.2**: Visualizing the URDF model in RViz2.
