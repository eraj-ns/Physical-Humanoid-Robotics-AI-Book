# ROS 2 General Sources

## Credible Sources for ROS 2 Concepts and Architecture

1.  **ROS 2 Documentation**: The official documentation is always the primary and most credible source for ROS 2. It provides comprehensive guides, tutorials, and API references.
    *   [https://docs.ros.org/en/humble/index.html](https://docs.ros.org/en/humble/index.html) (for Humble)
    *   [https://docs.ros.org/en/iron/index.html](https://docs.ros.org/en/iron/index.html) (for Iron)

2.  **ROS 2 Design Articles**: These articles provide deeper insights into the architectural decisions and underlying mechanisms of ROS 2.
    *   [https://design.ros2.org/](https://design.ros2.org/)

3.  **Open Robotics Blog**: The official blog often features updates, tutorials, and explanations of new features in ROS 2.
    *   [https://www.openrobotics.org/news/blog/](https://www.openrobotics.org/news/blog/)

4.  **The Construct (Robotics Academy)**: Offers many tutorials and courses on ROS 2, often with a focus on practical application. Their YouTube channel and website are excellent resources.
    *   [https://www.theconstructsim.com/robotics-developer-masterclass-ros2/](https://www.theconstructsim.com/robotics-developer-masterclass-ros2/)
    *   [https://www.youtube.com/c/TheConstructSim](https://www.youtube.com/c/TheConstructSim)

5.  **Books on ROS 2**: Several books have been published that provide structured learning paths for ROS 2.
    *   "ROS 2 in 7 Days" by Dr. S.M. Monzurul Islam (e.g., from Leanpub or similar platforms)
    *   "Hands-On ROS 2" by Patrick Goebel and Lentin Joseph (Packt Publishing)

These sources will be used to ensure the technical accuracy and depth of content in the module.
