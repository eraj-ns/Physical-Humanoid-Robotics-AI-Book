# Quickstart: Module 3

This guide provides a quick way to get started with the "AI-Robot Brain" module.

## Prerequisites

Before diving into this module, ensure you have a basic understanding of:

-   **ROS 2 Concepts**: Nodes, topics, services, and basic command-line tools. (Covered in Module 1).
-   **Linux Environment**: Familiarity with the Ubuntu terminal.
-   **Python**: Basic Python programming skills.

## Hardware Requirements

Many of the examples, especially those involving Isaac Sim and Isaac ROS, require a compatible NVIDIA GPU.

-   **Recommended**: NVIDIA RTX series GPU (e.g., RTX 30-series or 40-series).

## Software Requirements

-   **OS**: Ubuntu 22.04 LTS
-   **ROS 2**: ROS 2 Humble or Iron
-   **NVIDIA Isaac Sim**: Ensure you have followed the official installation instructions from NVIDIA.

## How to Read This Module

1.  **Start with the Introduction**: Begin with [Chapter 1: Introduction to the AI-Robot Brain](./01-Introduction-to-AI-Robot-Brain.md) to get a high-level overview of the concepts.
2.  **Follow the Chapters Sequentially**: The chapters are designed to be read in order, as they build upon each other.
3.  **Run the Code Examples**: To get the most out of the module, it is highly recommended to run the code examples in your own environment.

Ready to start? Begin with the [Introduction](./01-Introduction-to-AI-Robot-Brain.md).
