# URDF & Robot Modeling: Authoritative Sources

This document contains a curated list of primary and secondary sources for learning URDF.

## 1. Official Documentation

- **URDF Specification**: The official URDF specification.
  - [http://wiki.ros.org/urdf/XML](http://wiki.ros.org/urdf/XML)

## 2. Key URDF Tutorials

- **"Building a visual robot model from scratch"**: An official tutorial for building a URDF model.
  - [http://wiki.ros.org/urdf/Tutorials/Building%20a%20Visual%20Robot%20Model%20with%20URDF%20from%20Scratch](http://wiki.ros.org/urdf/Tutorials/Building%20a%20Visual%20Robot%20Model%20with%20URDF%20from%20Scratch)

## 3. Recommended Tools

- **SolidWorks to URDF Exporter**: A plugin for SolidWorks to export URDF files.
  - [http://wiki.ros.org/sw_urdf_exporter](http://wiki.ros.org/sw_urdf_exporter)
