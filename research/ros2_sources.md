# ROS 2 Fundamentals: Authoritative Sources

This document contains a curated list of primary and secondary sources for understanding the fundamentals of ROS 2.

## 1. Official Documentation

- **ROS 2 Docs (Primary)**: The official ROS 2 documentation is the most authoritative source.
  - [https://docs.ros.org/](https://docs.ros.org/)

## 2. Key Academic Papers

- **"Robot Operating System 2: Design"**: A paper from the original architects of ROS 2.
  - *Citation*: Macenski, S., et al. (2022). Robot Operating System 2: Design. *Science Robotics*, 7(68).

## 3. Recommended Textbooks

- **"Programming Robots with ROS"**: A comprehensive guide to ROS, with a second edition covering ROS 2.
  - *Citation*: Quigley, M., Gerkey, B., & Smart, W. D. (2015). *Programming Robots with ROS*. O'Reilly Media, Inc.
